GenerateObject
Generates an object for a given type. The type needs to be public, have a public default constructor and settable public properties/fields. Currently it supports the following types: Simple types: Int32, String, Enum, DateTime, Uri, etc. Complex types: POCO types. Nullables: NullableT. Arrays: arrays of simple types or complex types. Key value pairs: KeyValuePairTKey, TValue Tuples: TupleT1, TupleT1, T2, etc Dictionaries: IDictionaryTKey, TValue or anything deriving from IDictionaryTKey, TValue. Collections: IListT, IEnumerableT, ICollectionT, IList, IEnumerable, ICollection or anything deriving from ICollectionT or IList. Queryables: IQueryable, IQueryableT.
